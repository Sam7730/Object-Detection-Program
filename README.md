## 📌 Overview

AI Vision is a real-time object detection application that leverages the power of **YOLOv8** and **OpenCV** to identify and classify objects from a live webcam feed.

The system processes video frames in real time, detects supported objects, and displays bounding boxes with labels and confidence scores.

This project demonstrates practical applications of:

* Artificial Intelligence (AI)
* Machine Learning (ML)
* Deep Learning
* Computer Vision
* Real-Time Video Processing

## ⚠️ Known Limitations

The default YOLOv8 model is trained on the COCO dataset and supports 80 object classes.

Some objects such as:

* Glasses
* Playing Cards
* Credit Cards
* Student IDs
* Custom Items

may not be detected accurately because they are not included in the default training dataset.

Custom training can be performed to support additional object categories.
